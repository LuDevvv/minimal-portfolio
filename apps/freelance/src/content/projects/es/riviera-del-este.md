## Visión General

**Riviera del Este** representa la excelencia en el desarrollo inmobiliario digital. Diseñé esta plataforma de alto rendimiento para promocionar un complejo residencial de lujo en San Pedro de Macorís, enfocándome obsesivamente en la **conversión de visitantes a leads calificados**.

## El Desafío Técnico

El mercado de lujo exige perfección visual sin sacrificar velocidad. El reto era entregar una experiencia inmersiva rica en medios (videos, galerías de alta resolución) que cargara instantáneamente, incluso en conexiones móviles inestables.

### Objetivos Clave
- **Velocidad Elite**: Optimización agresiva de Core Web Vitals.
- **Internacionalización**: Experiencia nativa en Inglés y Español sin recargas.
- **Conversión**: Reducir la fricción entre el interés visual y el contacto.

## Mi Solución

Construí una SPA (Single Page Application) utilizando lo último en tecnología web moderna:

- **Next.js 15 (App Router)**: Aprovechando Server Components para renderizado ultrarrápido y SEO superior.
- **Optimización de Medios**: Integración con **Cloudinary** para servir imágenes AVIF/WebP adaptativas y carga progresiva de videos.
- **UI/UX Dinámica**: Animaciones sutiles con **Framer Motion** que guían la vista del usuario hacia los CTAs, y un sistema de filtros de propiedades intuitivo.
- **Formularios Inteligentes**: Integración con Resend API y validación Zod para asegurar leads limpios y notificaciones instantáneas a los agentes.

## Impacto en el Negocio

- **Retención**: El *smooth scrolling* y la carga rápida aumentaron el tiempo en página significativamente.
- **Captación**: El formulario de contacto con calendario inteligente optimizó la agenda de ventas, filtrando fechas no disponibles automáticamente.
- **Alcance Global**: La arquitectura multilingüe abrió el proyecto a inversores internacionales de manera efectiva.

### Stack Tecnológico
- **Core**: Next.js 15, TypeScript, Tailwind CSS 4
- **Internacionalización**: next-intl
- **Animaciones**: Framer Motion, Lenis Scroll
- **Backend/Servicios**: Resend API, Cloudinary
