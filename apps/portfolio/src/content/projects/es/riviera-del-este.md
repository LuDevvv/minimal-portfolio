## Visión General

**Riviera del Este** es una plataforma digital desarrollada para la promoción del complejo residencial en San Pedro de Macorís. El objetivo principal fue crear una herramienta efectiva para presentar el proyecto y facilitar el contacto con clientes interesados, priorizando la claridad y la facilidad de uso.

## El Desafío Técnico

El proyecto requería equilibrar una alta carga visual (imágenes y recorridos del inmueble) con un rendimiento óptimo en diversos dispositivos. Era necesario asegurar que la experiencia de navegación fuera fluida y que el contenido multimedia cargara eficientemente para no perder la atención del usuario.

### Objetivos Clave
- **Optimización de Carga**: Mantener buenos indicadores de Core Web Vitals.
- **Soporte Bilingüe**: Contenido accesible en Inglés y Español.
- **Flujo de Contacto**: Simplificar el proceso desde la visualización hasta el registro del interesado.

## Mi Solución

Desarrollé una aplicación web moderna enfocada en la usabilidad y el rendimiento:

- **Arquitectura Eficiente**: Uso de Next.js 15 para un renderizado rápido y amigable con motores de búsqueda.
- **Gestión Multimedia**: Implementación de **Cloudinary** para la entrega optimizada de imágenes y videos según el dispositivo del usuario.
- **Interfaz Interactiva**: Uso de **Framer Motion** para transiciones suaves que mejoran la navegación sin sobrecargar la experiencia visual.
- **Formularios Funcionales**: Sistema de contacto seguro con validación de datos y notificaciones automáticas vía Resend API.

## Impacto en el Negocio

- **Mejora en la Navegación**: La implementación de carga progresiva y navegación fluida ayuda a mantener el interés del usuario.
- **Gestión de Citas**: El formulario integrado facilita la coordinación inicial con el equipo de ventas.
- **Accesibilidad Internacional**: La opción multi-idioma permite presentar el proyecto a una audiencia más amplia de manera directa.

### Stack Tecnológico
- **Core**: Next.js 15, TypeScript, Tailwind CSS 4
- **Internacionalización**: next-intl
- **Animaciones**: Framer Motion, Lenis Scroll
- **Backend/Servicios**: Resend API, Cloudinary
